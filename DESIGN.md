# Shadow Covenant 3D — Game Design (run loop)

แนวเกม: 3D pixel 2.5D survivors-like (อิง Megabonk) ธีม gothic horror — ฮีโร่สายแสงสู้ฝูงผีสาง

## Core loop (1 รัน)
- เป้าหมาย: **ฆ่าบอสด่าน → เข้า portal → เคลียร์รัน** ให้ได้ **ภายใน 10 นาที** (เคลียร์สวย)
- **แท่นเรียกบอส + แท่นวาป = แท่นเดียวกัน** เกิด**สุ่มตำแหน่งในแมพทุกรอบ** อยู่ตั้งแต่ต้นรัน
  - เดินไปที่แท่น → **กดเรียกบอส (เรียกได้ตลอด เลือกจังหวะเอง)** → สู้บอส
  - ฆ่าบอส → **แท่นวาปที่จุดเดิมเปิด** → เข้าวาป = VICTORY / เคลียร์รัน
  - ยืดหยุ่น: เก่ง/พร้อม ก็ไปกดเรียกเร็ว เคลียร์ก่อน 10 นาที / อยากฟาร์มก่อนค่อยไปกด
- **Overtime (เกิน 10 นาที):** ยังไม่เคลียร์ → ศัตรู**โหดขึ้น + ออกเยอะขึ้นเรื่อยๆ ทุกนาที** แต่ **XP + ทองดรอปเยอะขึ้น** (high risk / high reward)
- ตาย = GAME OVER

## ความก้าวหน้าความยาก (ตามเวลา)
- 0–3 นาที: tier 0 (Bleakfield) เบาๆ ตั้งบิลด์
- 3–6 นาที: tier 1 (Fenmire) หนาขึ้น + elite ตัวแรก
- 6–9 นาที: tier 2 (Void Rift) หนัก + elite 2 ตัว
- 9–10 นาที: ฝูงไคลแม็กซ์
- 10 นาที+: overtime ramp (×ความโหด/ความถี่/รางวัล เพิ่มต่อเนื่อง)

## เศรษฐกิจ & สำรวจแมพ (เหตุผลให้ออกเดิน)
- **ทอง** = สกุลเงินหลัก เก็บจากมอน/หีบแตก
- **หีบ 3 ระดับ** (common / rare / epic) — **ใช้ทองเปิด ราคาแพงขึ้นเรื่อยๆ** ได้อาวุธ/passive/ไอเทม
- **ศาลเจ้า** — เรียก elite pack แลกรางวัลใหญ่ (เลือกท้าทายเอง)
- **NPC พ่อค้าสุ่ม** — โผล่สุ่มตำแหน่งบนแมพ เดินไปเจอ → เปิดร้านซื้อของด้วยทอง (อาวุธ/passive/ฮีล/รีโรล)

## ระบบที่ทำเสร็จแล้ว (ฐานพร้อม)
- เดิน/พุ่ง(dash+i-frame+เงาตามหลัง)/ชนสิ่งกีดขวาง/เนิน+ กล้องนิ่ง
- ฝูงมอน: separation, พฤติกรรม (chase/charger/shooter/exploder), อนิเมชันเดิน, knockback, เอฟเฟกต์ตาย
- อาวุธ 4 แบบ (bolt/spread/nova/orbit) — auto-attack
- เลเวลอัพเลือก tome/อาวุธ (เพดาน 3 อาวุธ: signature + 2)
- ตัวละคร 8 ตัว: signature weapon + base stats + passive โตตามเลเวล (ข้อมูลพร้อม รอสไปรซ์)
- หยุดเกม (ปุ่ม/P/Esc)

## สิ่งที่จะสร้างต่อ (เป็นเฟส)
**Phase 1 — โครงรัน (backbone):**
- ตัวจับเวลา + HUD เวลาเป้าหมาย 10:00
- แท่นเรียกบอส/วาป เกิดสุ่มตำแหน่งทุกรอบ (มีลูกศร/ไอคอนชี้) เดินไปกดเรียกได้ตลอด
- Boss fight: หลอดเลือด + แพตเทิร์นโจมตี (เฟส) → ฆ่า → แท่นวาปเปิด → เข้า = VICTORY screen
- Overtime escalation (โหด/ถี่/รางวัล ramp หลัง 10:00)

**Phase 2 — เศรษฐกิจ & สำรวจ:**
- หีบ 3 ระดับ (เปิดด้วยทอง ราคาขึ้นบันได)
- ศาลเจ้า (เรียก elite แลกรางวัล)
- NPC พ่อค้าสุ่ม (ร้านซื้อของ)

**Phase 3 — ความลึกบิลด์:**
- Weapon evolution / synergy (อาวุธ+passive ครบเงื่อนไข → ร่างพิเศษ)
- item/relic pool หลากหลายขึ้น + ระดับความหายาก

**ของเด่น Megabonk ที่พักไว้:** เนินสไลด์/กระโดด (verticality) — ค่อยพิจารณาภายหลัง
